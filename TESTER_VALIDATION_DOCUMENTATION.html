📋 Teknisk Information

  • Svarsalternativ-mappning: A=0, B=1, C=2, D=3, E=4, F=5
  • Keyboard shortcuts: A-F tangenter eller 1-6 nummer
  • Poängberäkning: (correctCount / totalQuestions) * 100
  • Rating-konvertering: Math.ceil(scoreRaw / 10) klampad till 1-10
  • Obesvarade frågor: Räknas som userAnswer = -1 (fel)
  • Session-binding: JWT med questionIds[] valideras på submit
  • GDPR: Data raderas automatiskt efter 12 månader
