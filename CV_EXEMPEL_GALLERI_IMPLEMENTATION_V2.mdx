## 🛠 TROUBLESHOOTING & COMMON ISSUES

### Build Errors

**Error**: `params is not a Promise`
```
Type 'Promise<{ yrke: string }>' is not assignable to type '{ yrke: string }'
```

**Solution**: Next.js 15 requires async params
```typescript
// ❌ Fel
export async function generateMetadata({ params }: { params: { yrke: string } }) {
  const data = exampleData[params.yrke]
}

// ✅ Rätt
export async function generateMetadata({ params }: { params: Promise<{ yrke: string }> }) {
  const { yrke } = await params
  const data = exampleData[yrke]
}
```

---

**Error**: SSG inte aktiverat (○ marker istället för ●)
```
○ /cv-exempel/[yrke]  (Static)
```

**Solution**: Verifiera att `generateStaticParams()` finns och returnerar alla yrken
```typescript
export async function generateStaticParams() {
  const yrken = Object.keys(exampleData)
  return yrken.map((yrke) => ({ yrke }))
}
```

---

**Error**: Schema.org validation errors (Google Search Console)
```
Missing field "name" (recommended)
Missing field "text" (recommended)
```

**Solution**: Filtrera tomma FAQ-entries
```typescript
// ❌ Fel - inkluderar tomma entries
mainEntity: faq.map(item => ({
  '@type': 'Question',
  name: item.fraga,  // kan vara tomt
  acceptedAnswer: {
    '@type': 'Answer',
    text: item.svar  // kan vara tomt
  }
}))

// ✅ Rätt - filter applicerad
mainEntity: faq
  .filter(item => item.fraga && item.svar && item.fraga.trim() !== '' && item.svar.trim() !== '')
  .map(item => ({
    '@type': 'Question',
    name: item.fraga,
    acceptedAnswer: {
      '@type': 'Answer',
      text: item.svar
    }
  }))
```
