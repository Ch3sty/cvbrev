# SEO-strategi: Galleri med Personliga Brev-exempel

**Författare:** SEO-teamet på jobbcoach.ai
**Datum:** 2025-11-22
**Baserad på:** Analys av 957 keywords från Google Keyword Planner

---

## Executive Summary

Keyword-datan visar en **massiv möjlighet** för jobbcoach.ai att dominera sökningar kring "personligt brev exempel [yrke]". Med totalt **12 100 sökningar/månad** enbart för huvudkeyword "personligt brev exempel" och hundratals yrkesspecifika long-tail varianter, kan ett strategiskt uppbyggt galleri generera **10 000-15 000 nya besökare/månad** inom 6-9 månader.

**Kritiska insikter:**
- **Undersköterska** är det överlägset största enskilda yrket (700+ sökningar/mån i olika varianter)
- **Lärare**, **student**, **ungdom**, **sjuksköterska** och **butik/säljare** är näst största kategorier
- Användare söker BÅDE "personligt brev exempel [yrke]" OCH "[yrke] personligt brev exempel"
- Keyword difficulty är **genomgående låg** (6-28) = stor rankingmöjlighet
- Starka "how-to" signaler: användare vill se konkreta exempel, inte bara mallar

### 5.2 FAQPage Schema


## 7. UNDVIKA DUPLICATE CONTENT

### Problem: Samma mall används för alla yrken

**Risk:** Google straffar duplicate content. Om alla exempel-sidor har identisk struktur och endast yrket ändras riskerar vi:
- Kannibaliserande rankings (sidorna konkurrerar mot varandra)
- Minskad crawl budget
- Lower domain authority

### Lösning: 4-stegs differentiering

#### 7.1 Unikt exempel för varje yrke
- Varje personligt brev-exempel MÅSTE vara 100% unikt skrivet
- Variera struktur: Vissa börjar med personlig anekdot, andra med kompetenssammanfattning
- Olika längd: Vissa yrken (chef, IT) kan ha längre brev (450 ord), andra kortare (ungdom, sommarjobb: 300 ord)

#### 7.2 Yrkesspecifik terminologi
- **Undersköterska:** "ADL-stöd", "personcentrerad vård", "palliativ vård", "journalsystem"
- **Lärare:** "formativ bedömning", "kunskapsmål", "elevhälsa", "klassrumsledning"
- **IT/Utvecklare:** "agila metoder", "CI/CD", "versionshantering", "tech stack"
- **Butikssäljare:** "merförsäljning", "kassahantering", "kundservice", "visual merchandising"

#### 7.3 Olika tips-sektioner
Anpassa "Tips för att skriva personligt brev som [yrke]" till varje bransch:
- Vård: Betona empati, teamarbete, stresshantering
- IT: Betona problemlösning, tekniska färdigheter, projektexempel
- Service: Betona kundkommunikation, flexibilitet, säljförmåga

#### 7.4 Unika FAQs per yrke
- **Undersköterska:** "Hur nämner jag licens/certifiering i personligt brev?"
- **Lärare:** "Ska jag nämna min undervisningsfilosofi?"
- **IT:** "Hur mycket tekniska detaljer ska jag inkludera?"

#### 7.5 Varierande ordräkning
- Strategiskt variera längd mellan 1,200-1,800 ord
- Vissa yrken (högre positioner) får djupare analys
- Undvik exakt samma antal ord på flera sidor

### 11.4 Hub-page med dynamisk listning

**I `/personligt-brev-exempel/page.tsx`:**

```typescript
import fs from 'fs';
import path from 'path';
import matter from 'gray-matter';
import Link from 'next/link';

export default async function PersonligtBrevExempelHub() {
  const contentDir = path.join(process.cwd(), 'content/personligt-brev-exempel');
  const files = fs.readdirSync(contentDir);

  const jobs = files.map(filename => {
    const filePath = path.join(contentDir, filename);
    const fileContent = fs.readFileSync(filePath, 'utf8');
    const { data } = matter(fileContent);

    return {
      slug: filename.replace('.mdx', ''),
      title: data.title,
      description: data.description,
      category: data.category,
      image: data.image
    };
  });

  // Gruppera efter kategori
  const jobsByCategory = jobs.reduce((acc, job) => {
    if (!acc[job.category]) acc[job.category] = [];
    acc[job.category].push(job);
    return acc;
  }, {});

  return (
    <div>
      <h1>Personligt Brev Exempel för alla yrken</h1>
      <p>Hitta inspiration och konkreta exempel på personligt brev för ditt yrke. Alla exempel är optimerade för ATS-system och modern AI-rekrytering.</p>

      {Object.entries(jobsByCategory).map(([category, categoryJobs]) => (
        <section key={category}>
          <h2>{category}</h2>
          <div className="job-grid">
            {categoryJobs.map(job => (
              <Link href={`/personligt-brev-exempel/${job.slug}`} key={job.slug}>
                <article>
                  <img src={job.image} alt={job.title} />
                  <h3>{job.title}</h3>
                  <p>{job.description}</p>
                </article>
              </Link>
            ))}
          </div>
        </section>
      ))}
    </div>
  );
}
```
