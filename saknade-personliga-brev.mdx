## Implementeringsriktlinjer

### Per nytt yrke behövs:

1. **Brevinnehåll (brevText):** 300-400 ord, 4-5 stycken
2. **seoIntro:** 200-300 ord, SEO-optimerad intro med \n\n mellan stycken
3. **varforDetFungerar:** 4-5 punkter med titel + beskrivning
4. **tips:** 4-5 tips med rubrik + text (längre tips med \n\n)
5. **faq:** 7-9 frågor med q + a
6. **relaterade:** 3-4 relaterade yrken (pekar på BEFINTLIGA slugs)

### Kvalitetskrav (från PERSONLIGT_BREV_GALLERI_IMPLEMENTATION.mdx):

- **Ton:** Mänskligt, inte AI-genererat
- **Inga tankstreck** (–) mellan fraser
- **Inga förbjudna ord:** kraftfull, innovativ, dynamisk, revolutionerande
- **Variera meningslängd** (5-25 ord)
- **Aktiv form:** "Jag ledde" inte "Projektet leddes"
- **Konkreta exempel** istället för vaga adjektiv
- **Kvantifierbara resultat** med kontext

### Filer att uppdatera per nytt yrke:

1. `src/app/personligt-brev-exempel/[yrke]/page.tsx` - Lägg till ny entry
2. `src/app/personligt-brev-exempel/page.tsx` - Uppdatera galleri-kort
3. `src/app/sitemap.ts` - Lägg till i PERSONLIGT_BREV_EXEMPEL array
