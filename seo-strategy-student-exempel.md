# SEO-strategi: Personligt Brev Exempel Student

## Executive Summary

**Mål:** Ranka topp 3 för "personligt brev student" inom 6 månader och generera 400-700 organiska besök/månad.

**Primärt keyword:** "personligt brev student" (estimerad sökvolym: 700/månad)

**Strategi:** Skapa den mest omfattande, praktiska och studentspecifika guiden på svenska marknaden – bättre än Monster, Indeed och CVmall.se.

**Innehåll:** 1700-2200 ord innehåll + interaktivt brevexempel med mall- och typsnittsval.

**ROI-potential:** 400-700 besök/månad × 5% konverteringsrate = 20-35 nya användare/månad från denna sida ensam.

## Schema Markup Strategy

### 1. Article Schema
**Syfte:** Hjälpa Google förstå att detta är en guide/artikel

```json
{
  "@type": "Article",
  "headline": "Personligt brev exempel för Student",
  "articleBody": "[Full brevtext]",
  "author": { "@type": "Organization", "name": "Jobbcoach.ai" },
  "datePublished": "2024-11-22",
  "dateModified": "2024-11-22"
}
```

**SEO-effekt:**
- Ökar chans för article-snippets i SERP
- Artikelbrödtext indexeras korrekt
- Datum visar freshness

### 2. FAQPage Schema
**Syfte:** Featured snippets för FAQ-frågor

```json
{
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Hur skriver jag personligt brev utan arbetslivserfarenhet?",
      "acceptedAnswer": { "@type": "Answer", "text": "[Svar]" }
    }
    // ... 7-9 frågor totalt
  ]
}
```

**SEO-effekt:**
- Featured snippets i SERP (Google visar FAQ-dropdown direkt i sökresultatet)
- Position zero-potential för 2-3 frågor
- Ökar CTR från SERP med 20-40%

**Prioriterade FAQ för featured snippets:**
1. "Hur skriver jag personligt brev utan arbetslivserfarenhet?" (högsta sökvolym)
2. "Hur lång ska ett personligt brev vara för student?"
3. "Ska jag nämna att det är sommarjobb jag söker?"

### 3. HowTo Schema
**Syfte:** Step-by-step guide för Google Assistant och voice search

```json
{
  "@type": "HowTo",
  "name": "Hur du skriver ett personligt brev som Student",
  "step": [
    { "@type": "HowToStep", "name": "Fokusera på överförbara färdigheter...", "text": "..." },
    // ... 5 steg (tips)
  ]
}
```

**SEO-effekt:**
- Voice search optimization ("Hur skriver jag personligt brev som student?")
- Möjlighet till HowTo rich snippet i SERP
- Google Assistant-integration

### 4. BreadcrumbList Schema
**Syfte:** Tydlig site-hierarki för Google

```json
{
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "position": 1, "name": "Hem", "item": "https://jobbcoach.ai" },
    { "position": 2, "name": "Personligt Brev Exempel", "item": ".../personligt-brev-exempel" },
    { "position": 3, "name": "Student", "item": ".../personligt-brev-exempel/student" }
  ]
}
```

**SEO-effekt:**
- Breadcrumb-navigation i SERP (ökar trovärdighet)
- Tydlig site-struktur för crawlers
- Stödjer intern länkning

## Risk Analysis & Mitigation

### Risk 1: Konkurrenter kopierar innehåll
**Sannolikhet:** Medel
**Impact:** Låg-medel

**Mitigation:**
- Vårt interaktiva exempel är svårt att kopiera (kräver utveckling)
- Kontinuerlig uppdatering av innehåll (refresh var 6:e månad)
- Bygg brand authority genom backlinks och social proof

### Risk 2: Google algorithm update
**Sannolikhet:** Hög (Google uppdaterar konstant)
**Impact:** Medel-hög

**Mitigation:**
- Fokusera på E-E-A-T (Experience, Expertise, Authoritativeness, Trust)
- Undvik over-optimization (naturligt språk, inte keyword stuffing)
- Diversifiera trafikskällor (inte bara SEO – även social, email, direct)

### Risk 3: Låg conversion från organisk trafik
**Sannolikhet:** Låg-medel
**Impact:** Hög (trafik utan conversion = slöseri)

**Mitigation:**
- A/B-testa CTA-placering och copy
- User testing för att förstå friction points
- Retargeting för användare som inte konverterade

### Risk 4: Innehållet åldras (blir irrelevant)
**Sannolikhet:** Hög (arbetsmarknaden ändras)
**Impact:** Medel

**Mitigation:**
- Sätt reminder att uppdatera innehåll var 6:e månad
- Övervaka trender i studentjobb (nya branscher, nya verktyg)
- Uppdatera datum i Schema markup för freshness signal
