🔍 Schema.org Validering

Använd Google's Rich Results Test för att validera Schema markup:

  1. Öppna en artikel (t.ex. CV Mall 2025)
  2. Högerklicka och välj "Visa sidkälla"
  3. Sök efter application/ld+json
  4. Verifiera att du ser:
    • BlogPosting schema med author Person
    • BreadcrumbList schema
    • FAQ schema (om artikeln har FAQ)
