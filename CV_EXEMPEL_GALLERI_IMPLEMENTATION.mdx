# CV-Exempel Galleri - Implementationsguide

## Översikt

CV-exempel galleriet är en motsvarighet till `/personligt-brev-exempel` men anpassat för CV-format. Systemet visar professionella CV-exempel för olika yrken med interaktiv mallförhandsvisning, SEO-optimerad copy och praktiska tips.

**Status**: Proof-of-concept slutfört med 1 yrke (Undersköterska)
**URL**: [https://jobbcoach.ai/cv-exempel](https://jobbcoach.ai/cv-exempel)
**Build Status**: ✅ SSG fungerar, 264 sidor genererade

---

## Arkitektur

### Filstruktur

```
src/app/cv-exempel/
├── page.tsx                        # Galleri-sida med alla CV-exempel
├── [yrke]/
│   ├── page.tsx                    # Data + SSG för specifikt yrke
│   ├── CVExempelPage.tsx           # Client component för presentation
│   └── InteractiveCVPreview.tsx    # Interaktiv mallförhandsvisning
```

### Key Differences från Personligt Brev-exempel

| Aspekt | Personligt Brev | CV-exempel |
|--------|-----------------|------------|
| **Preview** | HTML-genererad brevtext | SVG-baserade mallar (från `/mallar/*.svg`) |
| **Data Structure** | `exempelBrev.brevText` | `exempelCV` med strukturerad data |
| **Fokus** | Brevskrivning, tone-of-voice | Struktur, kvantifierbara resultat, ATS-optimering |
| **CTA** | "Skapa mitt personliga brev" | "Ladda ner CV-mallar" |
| **Färgschema** | Blue-purple gradient | Cyan-indigo gradient |

## Komponenter

### 1. Galleri-sida (`/cv-exempel/page.tsx`)

**Syfte**: Visa alla CV-exempel i ett filtrerbart galleri

**Features**:
- 10 yrken i proof-of-concept (kan skalas till 30+)
- Kategorisering: Vård, Teknik, Service, Utbildning, Ekonomi, Offentlig sektor
- Sökfunktion
- Featured examples (3 mest populära)
- FAQ-sektion
- CTA till `/verktyg/cv-mallar`

**Key Metrics**:
- Sökvolym visas per yrke
- "Populär" badge för top 3

### 2. CV-exempel sida (`/cv-exempel/[yrke]/page.tsx`)

**Syfte**: Data-lager med SSG-konfiguration

**Innehåller**:
- `exampleData` objekt med alla yrken
- `generateMetadata()` för SEO
- `generateStaticParams()` för SSG (viktig!)
- Default export som renderar `CVExempelPage`

**SSG-implementation**:
```typescript
// VIKTIGT: params är Promise i Next.js 15
export async function generateMetadata({ params }: { params: Promise<{ yrke: string }> }) {
  const { yrke } = await params
  // ...
}

export async function generateStaticParams() {
  const yrken = Object.keys(exampleData)
  return yrken.map((yrke) => ({ yrke }))
}

export default async function Page({ params }: { params: Promise<{ yrke: string }> }) {
  const { yrke } = await params
  // ...
}
```

### 3. CVExempelPage (`CVExempelPage.tsx`)

**Syfte**: Client-side presentation component

**Sektioner**:
1. **Hero**: Title, intro, CTA buttons
2. **SEO Intro**: Multi-paragraph fördjupad text
3. **Interactive CV Preview**: Mallförhandsvisning med växling
4. **Varför det fungerar**: 6 kort med principer
5. **Tips**: Expanderbara accordion-kort (6 tips)
6. **FAQ**: Expanderbara frågor (10 frågor)
7. **Relaterade exempel**: 3 andra yrken
8. **CTA**: Final call-to-action till CV-mallar

**Design**:
- Cyan-indigo färgschema
- Framer Motion animations
- Lucide icons
- Tailwind CSS

### 4. InteractiveCVPreview (`InteractiveCVPreview.tsx`)

**Syfte**: Visa CV i olika mallar (SVG-baserat)

**Key Features**:
- Template selector: Dropdown (desktop) / Bottom sheet (mobile)
- 8 CV-mallar från `SIMPLE_TEMPLATES` (src/lib/cv/simple-templates.ts)
- Visar SVG-bild från `/mallar/*.svg`
- Free vs Premium badge med Crown icon
- CTA till `/verktyg/cv-mallar` för nedladdning

**Tech Stack**:
- Next.js Image component för SVG rendering
- Framer Motion för transitions
- Click-outside detection (desktop only)
- Mobile-responsive med bottom sheet

**Templates användning**:
```typescript
import { SIMPLE_TEMPLATES } from '@/lib/cv/simple-templates'

// Templates innehåller:
// - id, name, description
// - imagePath: '/mallar/modern-minimal.svg'

// - tier: 'free' | 'premium'
```

## Skrivguide för CV-exempel

### Varför det fungerar (6 punkter)

**Syfte**: Pedagogisk, förklara VARFÖR exemplet är bra

**Format**:
```typescript
{
  rubrik: "Kort, påståendefull rubrik",
  text: "1-2 meningar förklaring. Ska vara konkret och educational."
}
```

**Teman att täcka**:
1. ATS-optimering (nyckelord, struktur)
2. Kvantifierbara resultat (siffror, mätbart)
3. Balans teknisk/mjuk kompetens
4. Certifieringar/trovärdighet
5. Profiltext som säljande sammanfattning
6. Progression/ansvar

### Tips (6 punkter)

**Syfte**: Actionable, praktiska råd för läsaren

**Format**:
```typescript
{
  rubrik: "Imperativ formulering - vad läsaren ska göra",
  text: "2-3 stycken (separerade med \n\n). Första stycket: varför. Andra stycket: hur. Tredje stycket: exempel."
}
```

**Teman**:
1. Branschspecifika nyckelord
2. Kvantifiera erfarenhet
3. Visa konkreta resultat (inte bara ansvar)
4. Anpassa profiltext per jobb
5. Lyft fram certifieringar
6. Balansera tekniska/mjuka färdigheter med bevis

**Tips-stil**:
- Inga tankstreck som skiljetecken
- Actionorienterat: "Inkludera...", "Visa...", "Anpassa..."
- Konkreta exempel: "Istället för X, skriv Y"
- Multi-paragraph för tips >80 ord (använd \n\n)

### FAQ (10 frågor)

**Syfte**: Long-tail keywords, vanliga user concerns

**Format**:
```typescript
{
  fraga: "Fråga formulerad som användaren skulle googla",
  svar: "Kort, rakt svar (2-3 meningar). Börja med 'Ja/Nej' om applicable."
}
```

**Teman att täcka**:
1. Hur långt ska CV vara?
2. Ska jag inkludera X? (t.ex. VFU, profilbild, certifieringar)
3. Hur visar jag Y? (t.ex. medicinsk delegering, stresstålig)
4. Hur anpassar jag för olika vårdmiljöer/branscher?
5. Vad gör jag om luckor i CV?
6. Hur mycket personlig info ska jag ha?
7. Ska jag lista alla arbetsgivare?
8. Hur lyfter jag fram teamarbete?
9. Branschspecifika frågor (t.ex. "Ska jag nämna truckkort?")
10. Format/struktur-frågor

## Troubleshooting

### Build errors

**Problem**: `params is not a Promise`
**Solution**: Next.js 15 kräver `params: Promise<{ yrke: string }>` och `await params`

**Problem**: SSG inte aktiverat
**Solution**: Verifiera att `generateStaticParams()` finns och returnerar alla yrken

### Preview issues

**Problem**: SVG templates visas inte
**Solution**: Kontrollera att `imagePath` i `SIMPLE_TEMPLATES` matchar filer i `/public/mallar/*.svg`

**Problem**: Mobile bottom sheet stängs inte
**Solution**: Verifiera click-outside detection är disabled på mobile (`!isMobile` check)
