## SVG REFERENCE

### Färger
```
blue:   #2563eb
red:    #dc2626
green:  #16a34a
black:  #000000
yellow: #ca8a04
purple: #9333ea
```

### Storlekar (Radie R)
```
small:  R = 20px
medium: R = 30px
large:  R = 40px
```

### Stroke Width
```
strokeWidth = Math.max(1, Math.round(R / 12))
small:  1.67px
medium: 2.5px
large:  3.33px
```

### Former - Koordinater

**Circle:**
```svg
<circle cx="50" cy="50" r="{R}" />
```

**Square (inskriven i cirkel):**
```javascript
halfSide = R / √2
x = 50 - halfSide = 50 - R/1.414
y = 50 - halfSide
width = halfSide * 2
```

**Triangle (liksidig, pekar uppåt):**
```javascript
top = (50, 50-R)
bottomRight = (50 + R*cos(30°), 50 + R*sin(30°))
bottomLeft = (50 - R*cos(30°), 50 + R*sin(30°))
```

**Diamond (romb):**
```javascript
top = (50, 50-R)
right = (50+R, 50)
bottom = (50, 50+R)
left = (50-R, 50)
```

**Hexagon (6-hörning):**
```javascript
6 punkter från -90° (top):
angle = -90° + (60° * i) för i=0..5
point = (50 + R*cos(angle), 50 + R*sin(angle))
```

**Star (5-taggig stjärna):**
```javascript
outer = R
inner = R * 0.5
10 punkter (alternerande outer/inner):
outerAngle = -90° + (72° * i)
innerAngle = outerAngle + 36°
```

### Patterns

**Striped (randigt):**
```svg
<pattern id="unique-id" patternUnits="userSpaceOnUse" width="6" height="6">
  <line x1="0" y1="0" x2="0" y2="6" stroke="{color}" stroke-width="1.5" />
</pattern>
```

**Dotted (prickigt):**
```svg
<pattern id="unique-id" patternUnits="userSpaceOnUse" width="6" height="6">
  <circle cx="3" cy="3" r="1" fill="{color}" />
</pattern>
```

**Crosshatch (korsschraffering):**
```svg
<pattern id="unique-id" patternUnits="userSpaceOnUse" width="6" height="6">
  <line x1="0" y1="0" x2="6" y2="6" stroke="{color}" stroke-width="1" />
  <line x1="6" y1="0" x2="0" y2="6" stroke="{color}" stroke-width="1" />
</pattern>
```

### Rotation
```svg
<g transform="rotate({degrees}, 50, 50)">
  {shape här}
</g>
```

Tillåtna rotationer: 0, 45, 90, 135, 180, 225, 270, 315 grader
